🔥 COMO JOGAR

🍾 1. O Giro da Sorte

A boca da garrafa deve parar exatamente apontada para o nome. Se parar "torta" entre dois jogadores, o sistema anula o giro e o próximo jogador gira.

🃏 2. Baralho Inteligente

Cada jogador tem 50 cartas. Ao ser escolhido, clique no seu monte para revelar uma [VERDADE] ou [DESAFIO].

Fases: Início (Leve) -> Meio (Esquenta) -> Final (Safadeza).

🔄 3. Regra do Azarado

Caiu 2x seguidas no mesmo lugar? O sistema ativa o Mico Automático. Não tem carta, é mico direto!

🏳️ 4. Botão Desistir

Ficou com medo? Clique em DESISTIR. Mas prepare-se: o sistema sorteará uma PUNIÇÃO obrigatória (Doses ou Micos piores).

💡 Dica de Ouro

"O que acontece no jogo, fica no jogo. Sejam sinceros e corajosos. A Heineken decidiu, está decidido!" 😈
